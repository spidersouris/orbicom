    C'est en effet ainsi qu'on écrivait les calendriers en Occident. Le lundi est le premier jour du lundii, le mardi est le second jour du lundi, le mercredi est le troisième jour du lundi, le jeudi est le quatrième jour du lundi, le vendredi est le cinquième jour du lundi, le samedi est le sixième jour du lundi, le dimanche est le septième jour du lundi.
    Le lundi était le premier jour de la semaine, le dimanche le septième. Ainsi, la semaine, en latin, était un septi-vega, un sept (sept) vêga (ve), un sept (sept) jours. L'espagnol, le portugais et les langues celtes, ont conservé le nom du premier jour de la semaine : « Lundy ».
