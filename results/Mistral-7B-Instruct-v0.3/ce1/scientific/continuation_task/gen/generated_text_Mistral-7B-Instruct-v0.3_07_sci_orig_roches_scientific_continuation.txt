Les pierres sont des roches dures qui ont souvent été utilisées pour construire des maisons. Les roches sont dures, et ils ne peuvent pas être facilement brisés.	 Cela signifie que les murs construits avec des pierres peuvent être très forts et durables.	 Il y a une autre sorte de roches que les pierres : les graviers. Il y a des petits rochers qui sont appelés des graviers. Ils peuvent être utilisés pour faire des chemins ou pour pavés les places de jeux.	 Les sables sont d'autres types de roches. Il y a des petits rochers qui se trouvent entre eux et ils sont appelés des sables. Les sables peuvent être utilisés pour faire des plages, ou pour remplir les trous.	 Les graviers et les sables sont souvent utilisés pour faire des bâtiments. Les bâtiments sont des structures construites par l'homme. Cela comprend les maisons, les écoles, les bureaux, les bibliothèques, etc.	 Les pierres, les graviers et les sables peuvent être trouvés dans de nombreux lieux. Les mines sont des endroits où l'on extrait les pierres, les graviers et les sables. Les carrières sont des endroits où l'on extrait les pierres. Les sables sont souvent extraits des rivières ou des plages. Les graviers peuvent être trouvés dans les ruisseaux ou les fleuves.
